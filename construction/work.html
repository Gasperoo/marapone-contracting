BLUEPRINT AUDITOR
FOR A RESIDENTIAL GC

The problem

A residential GC with 12–18 active projects was losing 6–8 hours per week manually cross-referencing blueprint revisions. Conflicts weren't caught until after trade mobilization — at cost.

What was built

A private Blueprint Auditor installed on a dedicated office machine. Ingests PDF blueprints, extracts dimension and scope data, cross-references revisions, and outputs a structured audit report — specific page, specific conflict, specific delta.

How it runs today

Project coordinator drops blueprints into a watched folder — report in shared drive within minutes. No cloud upload. Running independently for seven months.

FREIGHT INVOICE
INTELLIGENCE SYSTEM

The problem

A freight broker processing 300–400 invoices per month was spending five hours a week on manual validation — and still missing overcharges. One billing error went undetected for three months.

What was built

A Freight Invoice Intelligence system reading PDF and EDI invoices, extracting charge line items, and cross-referencing against rate confirmations. Discrepancies flagged in an exceptions report with exact variance amounts. Invoices never leave the premises.

How it runs today

Invoices land in a watched folder — exceptions report ready in two minutes. Ops team reviews flags, not every line. Running five months.

03
Trade & Logistics Northern Italy Professional Build

CUSTOMS COMPLIANCE
DOCUMENT VALIDATOR

The problem

An Italian trading company had recurring compliance errors on EU customs packages — wrong HS codes, missing certificates, invoice/packing list mismatches. Errors weren't caught until broker submission, with documents sometimes in three different languages.

What was built

A multilingual Customs Document Validator (Italian, English, German). Reads complete customs packages and validates for consistency, required fields, and HS code plausibility. Errors returned as a pre-submission checklist before anything goes to the broker.

How it runs today

Compliance officer runs packages before broker submission. Re-filing incidents dropped −78% in the first quarter. Deployed on internal server — no external API calls.
