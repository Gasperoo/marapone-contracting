BLUEPRINT AUDITOR
FOR A RESIDENTIAL GC

The problem

A residential GC with 12–18 active projects was losing 6–8 hours per week manually cross-referencing blueprint revisions. Conflicts weren't caught until after trade mobilization — at cost.

What was built

A private Blueprint Auditor installed on a dedicated office machine. Ingests PDF blueprints, extracts dimension and scope data, cross-references revisions, and outputs a structured audit report — specific page, specific conflict, specific delta.

How it runs today

Project coordinator drops blueprints into a watched folder — report in shared drive within minutes. No cloud upload. Running independently for seven months.

Construction GTA, Canada Commercial GC · Deployed

RFI & DAILY LOG
INTELLIGENCE STACK

The problem

A commercial GC running four mid-rise projects was tracking 80+ open RFIs across three project coordinators in three different spreadsheets. Daily logs were arriving as block-of-text emails from four superintendents — none in the same format, none searchable, none compliant with the OBC documentation requirement on two of the projects.

What was built

An RFI Backlog console plus a Daily Logs intelligence engine, both running on the office server. RFIs are auto-classified by trade, drawing reference, and priority on intake; logs are parsed into structured manpower / trades / delay records and rolled up into a weekly site brief. Both modules feed the same activity log so the PM team has a single source of truth.

How it runs today

Coordinators import the daily RFI CSV from email, supers paste shift logs in once a day — system handles the rest. Running for 4.5 months across all active projects. No subscription, no per-seat fee, no cloud egress.

03
Construction Southern Ontario Design-Build · Pilot Live

AI TAKEOFF + LIVE
COSTING PIPELINE

The problem

A residential design-build firm bidding 30–50 jobs a year was burning 6–10 hours per estimate on manual takeoff and another 2 hours pricing it against a supplier book that was 4–6 weeks stale. Estimators were leaving margin on the table on framing and structural items and overpricing finishes — the win rate had flatlined.

What was built

An Automated AI Takeoff module that extracts quantities directly from drawings — framing, sheathing, insulation, drywall, roofing, exterior, flooring, plumbing — paired with a live cost engine fed by an Ontario materials price feed updated daily. Outputs a fully labour-burdened estimate with overhead and markup ready to send to the client.

How it runs today

Estimator uploads drawings; live takeoff is back in under 20 minutes with priced bill of materials and benchmark check. System has been running on their on-prem build server for 3 months. Two more residential builders are on the install queue.
